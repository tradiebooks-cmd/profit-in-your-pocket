# Profit in Your Pocket — Tradie Books Australia

AI-powered PDF report analyser for tradies.

## Setup

1. Clone this repo
2. Run `npm install`
3. Create `.env.local` with your Anthropic API key:
   ```
   ANTHROPIC_API_KEY=sk-ant-...
   ```
4. Run `npm run dev` to test locally
5. Deploy to Vercel and add `ANTHROPIC_API_KEY` as an environment variable

## Deploying to Vercel

1. Push this repo to GitHub
2. Import on vercel.com
3. Add environment variable: `ANTHROPIC_API_KEY`
4. Deploy!
